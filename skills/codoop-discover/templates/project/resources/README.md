## 使用规则

- 新增资源前先检查是否已有可复用资源。
- 不在此目录存放用户上传内容。
- 未记录来源或许可的资源不得加入。
- 字体的视觉用途以 `design-system.md` 为准；本文件记录文件来源与许可。

## 新增资源流程

1. 放入最接近用途的现有或新建分组。
2. 在资源清单中登记用途、使用方、来源或许可。
3. 确认所有目标客户端都可正确使用该资源。
